# 2022年02月27日　(日)
お久しぶりです。

前回のポストからいろいろなことがあった。 なお残した葉は光沢じゃなかった。

フクマンギの葉が落ち続け、Redditで問題を聞きました。　やっぱり、[さっきに書いたのように](2021-12-29.md)、　人工照明は盆栽のために全然足りないようです。
